## Descriçao de execuçao questao_02

#### Requisitos 
- Maven
- Docker
- Docker Compose

#### Construçao
- Execute `mvn clean package` na raiz do diretotio **questao_01** para compilar e empacotar todos os arquivos **.jar**.

#### Execuçao 
- Execute`docker-compose up --build` para rodar todos os containers.
